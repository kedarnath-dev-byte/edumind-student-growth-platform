# Pilot Safety And Parent Consent

## A. Purpose

The EduMind pilot should help students build learning habits safely.

The pilot focuses on:

- Daily learning reflection
- Honest confusion sharing
- Scientific revision
- Memory Rescue
- Peer learning
- Parent and teacher support

The goal is learning improvement and feedback, not comparison or public
performance pressure.

## B. Safety Principles

- Do not publicly display full student names.
- Do not publicly display marks.
- Do not publicly shame students.
- Do not use ranking or leaderboards during the pilot.
- Hide faces or get explicit parent permission before showing faces.
- Avoid showing private dashboard data in social media posts.
- Use "pilot students" or "early learning families," not "customers" unless
  they are actually paying.
- Use supportive language only.
- Treat "I need support" as a useful learning signal, not a problem with the
  child.

## C. Parent Consent Message

WhatsApp-ready message:

```text
Hi everyone, I am starting a small EduMind pilot with our students.

EduMind helps children build learning habits: daily learning reflection, revision, memory rescue, and peer help.

For safety, I will not publicly show children’s full names, private learning details, marks, or faces without permission. If we record any video, we can show only hands/mobile screens or blur faces.

This pilot is for learning improvement and feedback, not for comparison or ranking.

Thank you for trusting and supporting this village learning initiative.
```

## D. Video Recording Rules

- Prefer hands/mobile screen only.
- Blur faces if needed.
- Do not show student names.
- Do not show private learning struggles without permission.
- Ask parents before recording.
- Avoid public marks comparison.
- Do not claim paid customers unless they are actually paid.
- Keep videos short, respectful, and focused on learning habits.

## F. Village Pilot Checklist

Before pilot:

- Backend running
- Frontend running
- Demo data or real users ready
- Parent permission confirmed
- No private data shown publicly
- Weekly metrics noted
- Feedback collected
- Students understand that the pilot is not a marks competition
- Parents understand that honest confusion is a support signal
