# Mobile Install Guide

## A. Purpose

EduMind can be opened from a mobile browser and added to the phone home screen.
This is the safest first mobile step before Android APK or iPhone native app
work.

## B. Android Chrome Steps

1. Open the EduMind Vercel URL in Chrome.
2. Tap the three-dot menu.
3. Tap **Add to Home screen** or **Install app**.
4. Confirm the EduMind icon appears on the home screen.

## C. iPhone Safari Steps

1. Open the EduMind Vercel URL in Safari.
2. Tap the Share button.
3. Tap **Add to Home Screen**.
4. Confirm the EduMind icon appears on the home screen.

## D. Important Notes

- Internet is required for now.
- Login/auth is coming later.
- Do not share private student screenshots publicly.
- If the backend sleeps on Render, the first load may take some seconds.
